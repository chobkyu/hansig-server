users
ㄴ 회원가입 시
    /post user 
        ㄴ 성공 시 201로 응답
        ㄴ 입력이 잘못 되었을 경우 400으로 응답
        ㄴ 중복값이 발생했을 경우 409로 응답한다
        


ㄴ 로그인
    /post login
        ㄴ 성공 시 201로 응답, 토큰 반환
        ㄴ 입력 값이 잘못 되었을 경우 400으로 응답
        

ㄴ 유저 정보 수정 시
    /post user/info
        ㄴ 성공 시 201로 응답
        ㄴ 입력 값이 잘못 되었을 경우 400으로 응답
        ㄴ 로그인이 안되어 있을 시 401로 응답
    ㄴ 비밀번호 변경 시
        ㄴ 성공 시 201 리턴
        
ㄴ 유저 정보 읽을 시
    /get user/info
        ㄴ 성공 시 200 리턴
        ㄴ 유저 없을 시 404 리턴
        ㄴ 잘못된 입력일 시 400
        ㄴ 중복 값이 발생했을 경우 409
